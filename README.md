# NASA Space Apps Challenge 2022 – DEEPTHINKING

## Descripción general

Este repositorio contiene el trabajo desarrollado por el equipo **DEEPTHINKING** durante el **NASA Space Apps Challenge 2022**, específicamente para el desafío **Carrington Event**.

El proyecto fue **ganador a nivel regional**, y aborda el análisis y comprensión de eventos solares extremos y su impacto potencial sobre la Tierra, combinando análisis de datos, visualización y razonamiento científico.

* Página oficial del proyecto en NASA Space Apps:
  [https://2022.spaceappschallenge.org/challenges/2022-challenges/carrington-event/teams/deepthinking/](https://2022.spaceappschallenge.org/challenges/2022-challenges/carrington-event/teams/deepthinking/)

## Objetivo del proyecto

El objetivo principal fue explorar y analizar información relacionada con el **Evento Carrington**, el evento solar más intenso registrado, con el fin de:

* Comprender sus causas y efectos.
* Analizar datos relevantes asociados a tormentas solares.
* Generar visualizaciones y conclusiones que faciliten la divulgación científica y la concienciación sobre riesgos tecnológicos actuales.

## Contenido del repositorio

El núcleo del proyecto se encuentra en el notebook:

* `SpaceApps_DEEPTHINKING.ipynb`

Este notebook incluye:

* Análisis exploratorio de datos relacionados con actividad solar.
* Procesamiento y limpieza de datos.
* Visualizaciones para interpretar fenómenos solares extremos.
* Explicaciones y conclusiones integradas directamente en el flujo del análisis.

## Tecnologías utilizadas

* **Python**
* **Jupyter Notebook**
* Librerías comunes de análisis de datos y visualización (NumPy, Pandas, Matplotlib, entre otras)

## Contexto del desafío

El desafío **Carrington Event** planteó la necesidad de reflexionar sobre cómo un evento solar extremo similar al ocurrido en 1859 afectaría a la infraestructura moderna:

* Redes eléctricas
* Satélites
* Sistemas de comunicación
* Infraestructura tecnológica crítica

Este proyecto se enfoca en el análisis conceptual y de datos como base para la comprensión de estos riesgos.

Proyecto desarrollado en el marco del **NASA Space Apps Challenge 2022**, obteniendo **primer lugar a nivel regional**.
